Variablen

Datentyp Erklärung Beispiel-Code

x = "Hallo"		 Speichert Text, auch "String" genannt. Ein String steht immer in Anführunszeichen. name = "hanspeter"

end
Tabelle:

kaufen = {"brot", "nudeln", "Tee"}		 Speichert mehrere, auch verschiedende, Daten in einer Tabelle. Man kann die Daten lesen, wenn man den Namen der Tabelle gefolgt von der Zahl füe den gewünschten Eintrag eckigen Klammern schreibt: kaufen[2] gibt "nudeln" zurück. kaufen = {"brot", "nudeln", "tee"}
function _draw()

 print(kaufen[2],45,67,7)

end

Sonderfunktionen (in diesen Funktionen hier steck dein gesamter Code drin)

Funktion Erklärung Beispiel
_init() Wird nur einmal ausgeführt, wenn das Spiel gestartet wird. Hier kann Variablen definieren und alles für den Start des Spiels einrichten. function _init()
  x = 20
  y = 50
  r = 30
end
_update() Wird 30 mal pro Sekunde ausgeführt. Alles was in dieser Funktion steht wird also unendlich mal wiederholt (loop). Hier werden z.B. die Bewegunden des Spiels berechnet. Sie wird jeweils vor der _draw()-Funktion ausgeführt. function _update()
  if btn(1) then x = x + 1 end
end
_draw() Wird 30 mal Pro Sekunde wiederholt. Alles was in dieser Funktion steht wird also unendlich mal wiederholt (loop). Hier werden alle Befehle ausgeführt, die den Bildschirninhalt verändern. Sie wird nach der _update()-Funktion ausgeführt. function _draw()
  cls(12)
  circ(x,y,r)
end

Zeichnet die Map		 x: X-Position auf der Map (linke obere Ecke) in Tiles.

Ein Tile ist so groß wie ein 8x8 Sprite.		 function _draw()

end
camera([x,y])

Bewegt die Kamera über die Map		 x: Position auf der X-Achse

pset(x,y,[c])

Setzt einen Pixel

x: Position auf der X-Achse
y: Position auf der Y-Achse
c: Farbe (color)

function _draw()
 cls()
 pset(30,40,7)
end

line(x0,y0,x1,y1,[c])

Zeichnet eine Linie

x0: X-Position Anfang
y0: Y-Position Anfang
x1: X-Position Ende
x2: Y-Position Ende
c: Farbe

function _draw()
 cls()
 line(8,8,120,120,7)
end

circ(x, y, r, [c])
circfill(x, y, r, [c])

Einen Kreis zeichnen

x: Position auf der X-Achse
y: Position auf der Y-Achse
r: Radus
c: Farbe (optional)

cls()
circ(40,40,25,6)
circfill(90,70,20,6)

rect(x0,y0,x1,y1,[c])
rectfill(x0,y0,x1,y1,[c])

x0: X-Position linke obere Ecke
y0: Y-Position linke obere Ecke
x1: X-Position rechte obere Ecke
y1: Y-Position rechte obere Ecke

cls()
rect(20,40,50,60,6)
rectfill(80,60,120,90,6)

spr(n,x,y,[w,h],[flip_x],[flip_y])

Ein Sprite zeichnen

n: Spritenummer
x: Postion auf der x-Achse
y: Position auf der y-Achse
w: Breite in 8x8 Sprites (z.B. 2=16 Pixel)
h: Höhe in 8x8 Sprites (z.B. 2=16 Pixel)
flip_x: 0 oder 1, spiegelt X-Achse
flip_y: 0 oder 1, spiegelt Y-Achse

cls()
spr(2, 48, 48, 2, 2)

Mathe

Funktion Argumente Beispiel Input Beispiel Output
rnd(x)

Erzeugt eine Zufallszahl		 x: Die Höhe der Zufallszahl. 10 gibt z. B. eine zufällige Zahl zwischen 0 und 9
flr(x)

Rundet, indem sie alle Nachkommastellen abschneidet		 x: Die Zahl die gerundet wird
abs(x)

Nimmt den Betrag einer Zahl (Abstand von 0)		 x: Die Zahl deren Betrag ermittelt werden soll
sin(x) cos(x)

Sinus und Cosinus		 x: die Zahl deren Sinus oder Cosinus gegeben wird, wobei 1 eine ganze Umdrehung bedeutet cnt=0 --counter
s=0 --sinus
c=0 --cosinus

function _draw()
      cls()
      cnt+=0.005
      s = sin(cnt)
      c = cos(cnt)
      circfill(s*48+64,c*48+64,8,6)
end

Advanced
sspr( sx, sy, sw, sh, dx, dy, [dw,] [dh,] [flip_x,] [flip_y] )

Zeichnet ein Rechteck von Pixeln,

Draws a rectangle of pixels from the sprite sheet, optionally stretching the image to fit a rectangle on the screen.		 sx: X-Koordinate der linken oberen Ecke des ausgewähltens Sprites
sy: Y-Koordinate der linken oberen Ecke des ausgewähltens Sprites
sw: Breite des ausgew. Sprites
sh: Höhe des ausgew. Sprites
dx: X-Koordinate der linken oberen Ecke des Sprites auf dem Bildschirm.
dy: Y-Koordinate der linken oberen Ecke des Sprites auf dem Bildschirm.
dw: Breite des gezeichneten Sprites
dh: Höhe des gezeichneten Sprites
flip_x: Wenn wahr (1) wird das Sprite entlang der X-Achse gespiegelt
flip_y: Wenn wahr (1) wird das Sprite entlang der Y-Achse gespiegelt		 -- draw the 8 x 8 image from (8, 0) at screen location (60, 60)
sspr(8, 0, 8, 8, 60, 60)

-- draw the same image but stretched to 12 x 20 at screen location (44, 48)
sspr(8, 0, 8, 8, 44, 48, 12, 20)
